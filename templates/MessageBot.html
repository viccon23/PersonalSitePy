ChatGPT Discord Bot

In this small personal project, I wanted to create a bot on discord that would utilize the ChatGPT API and respond to user questions that would be asked on a channel in a Discord Server. The bot remembers the previous 15 user responses so that context will remain and preserve conversation between the user and the bot. The bot will also differentiate by user, so if one user is conversating with the bot and another user types another question, then the bot will remember any previous messages from that specific user, and not from the other. In short, it does one-on-one conversations.
The project was also a great way to become introduced to using Node.js, Javascript, and discord.js.
