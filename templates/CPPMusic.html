C++ Music Playlist Management System

This code implements a C++ program that simulates a playlist management system. It allows users to create a playlist, add tracks to the playlist, remove tracks, toggle track favorites, insert tracks at specifiic positions, display the playlist, and search for tracks by genre. The program utilizes various data structures such as linked lists to manage the playlist and song information. It also includes a menu-driven interface for user interation.
